Description

A module using this behaviour can be used by supercast to authenticate clients. The return will be used by the beha_supercast_acctrl module used by the application

The only function to be exported is authenticate/2. authenticate(Uname, UPass) -> Any::term() | fail

Note that the returned term() can be anything but must be understandable by the beha_supercast_acctrl module used by the application.
